First time setup — create your own endpoint
＋

Only needed if you are the first person in your household to set up sync. Everyone else just pastes your URL above.

This creates a Google Sheet in your own Google Drive that acts as the shared database. You run this once, then share the resulting URL with your household members.
1
Go to script.google.com, create a new project, select all existing code and delete it, then paste the Apps Script code below.
2
Click Deploy → New deployment. Click the gear icon and select Web app. Set "Execute as" to Me and "Who has access" to Anyone. Click Deploy and complete the authorization.
3
Copy the Web App URL and paste it into the field above. Tap Save endpoint. Share that same URL with your household members — they only need to paste it above, nothing else.
